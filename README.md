# まるのホームページ作成ガイド

このリポジトリには、名古屋在住の投資家・クリエイター **まる** さん向けの
パーソナルホームページのサンプルが含まれています。YDK の名称を出さない
「個人感」重視のデザインと、note や Shopify、インスタグラム、Threads
など外部サービスを埋め込む機能、勉強会申込フォーム、日経平均株価や
ドル円レート、日本10年債利回りなどの指数表示が実装されています。

## 構成

* `app.py` – Streamlit で構築されたマルチページ Web アプリケーション。左側の
  サイドバーでメニューを切り替えながら、各セクションを閲覧できます。
* `signup_data.json` – 勉強会申込フォームの送信内容をローカルに保存するファイル
  (初回は自動生成されます)。

## インストールとローカル実行

Python 3.9 以上の環境を想定しています。まず必要なライブラリをインストールします。

```bash
pip install streamlit requests
```

次にアプリケーションを起動します。

```bash
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501` にアクセスするとサイトを閲覧できます。

## 外部サービスの埋め込み方法

### note

note 記事をサイトに表示するには、**iframe 埋め込み**と**RSS 連携**の2通りがあります。以下は記事を iframe で埋め込む方法です。

1. note の記事ページで「サイトに貼る」を選択すると、埋め込み用の HTML コードが発行されます【597274875667231†screenshot】。
2. `app.py` の「note ブログ」ページに表示される入力ボックスに、自分の note 記事 URL を入力します。
3. Streamlit アプリが記事を iframe で表示します。

RSS 連携を利用すると、マガジンまたはクリエイターの**記事一覧**を埋め込むことができ、記事の追加・編集も自動的に反映されます。STUDIO などのサイトビルダーでは、RSS 連携によりサイトの UI に沿ってデザインをカスタマイズすることが可能です【597274875667231†screenshot】。

### Shopify（Buy Button）

Shopify の *Buy Button* を使うと、WordPress や静的サイトでも商品を販売できます。使い方は次の通りです。

1. Shopify 管理画面で Buy Button 販売チャネルを追加し、Buy Button を作成します【516483693416963†L167-L184】。
2. 商品単位またはコレクション単位でボタンを生成し、色やレイアウトをカスタマイズします【516483693416963†L169-L176】。
3. 「Next」をクリックすると埋め込みコードが自動生成されるのでコピーして、`app.py` の「オンラインショップ」ページ内のテキストエリアに貼り付けると商品ボタンが表示されます【516483693416963†L296-L304】。

### Instagram

公開設定の Instagram 投稿やプロフィールは公式機能で埋め込み可能です。手順は次の通りです【628245861234190†screenshot】。

1. コンピュータから Instagram ウェブサイトを開き、埋め込みたい投稿またはプロフィールにアクセスします。
2. 投稿右上の三点アイコンをクリックし「Embed」を選択します。
3. 表示されるウィンドウで「Copy Embed Code」を押すとコードがコピーされます。
4. コードを `app.py` の「Instagram」ページに貼り付ければ表示されます。

### Threads

Threads の投稿は、投稿右上のメニューから **Get embed code** を選択することで取得できます。SociableKIT による手動埋め込み手順は次の通りです【351750113714182†L260-L276】。

1. 埋め込みたいプロフィールまたは投稿ページにアクセスします。
2. 投稿右上の三点メニューをクリックし「Get embed code」を選びます。
3. 表示された embed コードをコピーして、`app.py` の「Threads」ページに貼り付けます。

### 勉強会申し込みフォーム

アプリには簡易的な参加申込フォームが用意されています。送信されたデータは `signup_data.json` に保存され、後から確認できます。実運用では Google Forms など外部ツールを使って埋め込むこともできます。Google Forms を埋め込む場合の手順は以下の通りです【347123601776776†L69-L80】【347123601776776†L108-L159】。

1. Google Forms でフォームを作成し、「Send」をクリックします。
2. 「Embed (<> アイコン)」を選び、表示された HTML コードをコピーします。
3. サイトの適当な場所に iframe として貼り付ければフォームが表示されます。

## 市場指数の表示

`app.py` では、日経平均株価、ドル円為替レート、日本10年債利回りを取得する関数が用意されています。取得に必要な API キーは環境変数として設定します。

* **Nikkei 225:** FRED のシリーズ `NIKKEI225` から最新の終値を取得します。FRED の API キーを `FRED_API_KEY` に設定してください。Nikkei 225 のデータは FRED が提供しており、例えば 2026 年 2 月 6 日の終値は 54,253.68 でした【344396590486891†L78-L96】。
* **日本10年債利回り:** FRED シリーズ `IRLTLT01JPM156N` を使用しています。データは月次で、2025 年 12 月の利回りは 2.06% でした【962069075661185†L80-L98】。
* **ドル円為替:** Alpha Vantage の `CURRENCY_EXCHANGE_RATE` API を利用します。無料の API キーを `ALPHAVANTAGE_API_KEY` に設定してください。レート制限に注意してください。

API キーを環境変数に設定すると、自動的に最新データが表示されます。設定しない場合は `None` が表示されます。

### Streamlit Community Cloud

Streamlit の公式無料ホスティングを利用する方法です。GitHub リポジトリに `app.py` を配置し、Streamlit の Community Cloud に接続するだけでデプロイできます。個人アプリの利用には十分ですが、月間稼働時間に制限があります。

## 静的サイトバージョン

Streamlit ではなく静的な HTML/CSS/JavaScript で構築したい場合は、以下のような構成が考えられます。

1. **index.html** – 各セクションを `<section>` 要素で分け、note や Shopify の埋め込みコードを直接記述します。
2. **style.css** – フォントや色、レイアウトを定義して、洗練されたデザインを実現します。例えば pastel 系の背景色やカード風デザインを採用すると柔らかい印象になります。
3. **script.js** – 必要に応じて JavaScript を記述し、API から指数を取得して表示します。Python を使用する場合はバックエンド (Flask など) を立ち上げ、`fetch` API で JSON を読み込むようにします。

静的サイトは Netlify や Vercel、Cloudflare Pages などの無料プランで手軽に公開できます。
